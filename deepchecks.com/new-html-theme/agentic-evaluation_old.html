Your Tool-Calling Agents

How It Works Book a Demo

Integrations

Agentic Workflow

Prompt

The user or system
instruction that kicks off an
agent session

Planning

Agent’s internal reasoning
step where it breaks the
task into sub-steps

Tool Call

LLM issues a function / API
call to retrieve or update
external data

Response

LLM uses the tool output
plus its context to generate
the final results

End-to-End Evaluation

Treat a full agent workflow (including agent
planning, action, and response) as a session, not
just a single interaction.

  • Auto-scoring by determining when to use
    each evaluation agent
  • Create customized evaluation agents
  • Supports auto annotations for Agent sessions
  • Compare versions of prompts, agents &
    models

Swarm of Evaluation
Agents

The algorithmic backbone of Deepchecks is a set
of small language models (SLMs) & multi-step
NLP pipelines, that work together as a swarm
using Mixture of Experts (MoE) techniques to
simulate an intelligent, human, annotator.

  • Hallucination detection experts
  • Planning accuracy evaluators
  • Rule-based checks for objective criteria

Compare Agentic
Workflows

Side-by-side comparisons of different agents,
models, or configurations.

  • Head-to-head agent version comparison
  • A/B testing of different agent strategies
  • Model performance benchmarking
  • Configuration effectiveness scoring
